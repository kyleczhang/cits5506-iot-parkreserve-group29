# cits5506-iot-parkreserve-group29

## Project Title

ParkReserve: IoT-Based Parking Reservation with Automated Barrier Control

## Project Description

This project proposes an IoT based smart parking monitoring and reservation system designed to improve parking efficiency and provide real time information to drivers. In many parking areas, drivers spend unnecessary time searching for available spaces because the occupancy of parking spots is not clearly visible. Visual indicators can help drivers identify whether a spot is available from a distance without needing to drive up to each space.

Each parking spot will be equipped with a sensor, such as an ultrasonic or infrared sensor, to detect the presence of a vehicle. The sensor sends data to a computing node, such as an ESP32 microcontroller, which processes the parking status and updates an indicator light from green (available) to red (occupied). The computing node communicates with a web dashboard using WiFi, allowing users to check parking availability through a mobile device or web interface.

A key feature of the system is its reservation function. Users can reserve an available parking spot online, after which the system changes the indicator to yellow to show the space is reserved. To make the reservation more visible and enforceable in the physical environment, a small barrier can also rise automatically to mark the reserved spot. This combines real time monitoring, user interaction, and physical actuation in one IoT system.
